# Cursor Landing Page Clone

A pixel-perfect recreation of the [Cursor.com](https://cursor.com) landing page built with pure HTML and CSS. This project demonstrates advanced CSS techniques, modular architecture, and attention to design details.

## Live Demo

**[View Live Site](https://swapnilsanghvi.github.io/cursor-lp-clone/)**

## Sections Implemented

- Navigation bar with logo and CTA buttons
- Hero section with headline and product screenshot
- Trusted by section with company logos
- Feature sections with alternating layouts
- Feature cards grid
- Testimonials with quotes and avatars
- Changelog updates
- Team section
- Final call-to-action
- Footer with multi-column links

## Design System

### Colors

```css
--color-bg-primary: #14120B    /* Dark brown background */
--color-text-primary: #EDECEC  /* Off-white text */
--color-text-secondary: #8E8E8E /* Gray text */
--color-accent: #F54E00        /* Orange accent */
```

### Typography

- **Font Family**: CursorGothic with system fallbacks
- **Base Size**: 1rem = 10px (using 62.5% root font-size)
- **Headings**: 400-500 weight
- **Body**: 400 weight

### Layout

- **Container Width**: 1300px
- **Spacing**: Rem-based units for consistency
- **Grid**: CSS Grid for cards and layouts

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools required!

### Installation

1. Clone the repository
```bash
git clone https://github.com/SwapnilSanghvi/cursor-lp-clone.git
```

2. Navigate to the project directory
```bash
cd cursor-lp-clone
```

3. Open `index.html` in your browser
```bash
open index.html
```

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with custom properties
- **CSS Grid** - Layout system
- **Flexbox** - Component alignment
- **CSS Custom Properties** - Theming and consistency

## Key Techniques

### Modular CSS Architecture
Each section has its own CSS file, making the codebase maintainable and scalable.

### Rem-based Units
All measurements use rem units with a 10px base for easy calculation and accessibility.

### CSS Custom Properties
Centralized color and layout values for consistent theming.

### Semantic HTML
Proper use of HTML5 elements for better accessibility and SEO.
